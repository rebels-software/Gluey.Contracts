Payload: [0xFF, 0x64,0x00, 0xC8,0x00, 0x2C,0x01] = header=255, readings=[100, 200, 300] LE

**Contract 2: Fixed scalar array big-endian**

Payload: [0xFF, 0x00,0x64, 0x00,0xC8, 0x01,0x2C]

**Contract 3: Semi-dynamic scalar array**
```json
{
  "kind": "binary",
  "endianness": "little",
  "fields": {
    "errorCount": { "type": "uint8", "size": 1 },
    "codes":      { "dependsOn": "errorCount", "type": "array", "count": "errorCount", "element": { "type": "uint8", "size": 1 } },
    "trailer":    { "dependsOn": "codes", "type": "uint16", "size": 2 }
  }
}
```
Payload for 2 codes: [0x02, 0x0A, 0x14, 0xD2,0x04] = count=2, codes=[10,20], trailer=1234

**Contract 4: Zero-count semi-dynamic**
Same contract as #3, payload: [0x00, 0xD2,0x04] = count=0, codes=[], trailer=1234

**Contract 5: Truncated payload**
Same contract as #3, payload: [0x05, 0x0A, 0x14, 0x1E] = count=5 but only 3 bytes for elements -- parse 3 elements

**Contract 6: Fixed struct array**
```json
{
  "kind": "binary",
  "endianness": "little",
  "fields": {
    "errorCount": { "type": "uint8", "size": 1 },
    "errors":     { "dependsOn": "errorCount", "type": "array", "count": 2, "element": { "type": "struct", "size": 3, "fields": {
      "code":     { "type": "uint8", "size": 1 },
      "severity": { "dependsOn": "code", "type": "uint16", "size": 2 }
    }}}
  }
}
```
Payload: [0x02, 0x01,0xF4,0x01, 0x02,0xE8,0x03] = errorCount=2, errors=[{code:1,severity:500},{code:2,severity:1000}]
