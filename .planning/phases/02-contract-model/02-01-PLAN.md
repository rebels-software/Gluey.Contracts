    internal sealed class BinaryContractNode

        internal ArrayElementInfo? ArrayElement { get; init; }
        internal Dictionary<string, string>? EnumValues { get; init; }
        internal string? EnumPrimitive { get; init; }
        internal object? Count { get; init; }  // int for fixed, string for semi-dynamic

        // Validation rules
        internal ValidationRules? Validation { get; init; }

        // Extensions
        internal SchemaErrorInfo? ErrorInfo { get; init; }
        internal string? XDescription { get; init; }

        // Display

        // Struct sub-fields (for array elements with type "struct")
        internal BinaryContractNode[]? StructFields { get; init; }
    }

    **9. Create BinaryContractLoader (src/Gluey.Contract.Binary/Schema/BinaryContractLoader.cs):**
    Internal static class with:
    ```csharp
    internal static (Dictionary<string, BinaryContractNode>? Fields, string? ContractEndianness, ContractMetadata? Metadata) Load(
        ReadOnlySpan<byte> utf8Json, ErrorCollector errors)
    ```
    Steps:
    1. Deserialize with JsonSerializer.Deserialize<ContractDto>(utf8Json, s_options) in try/catch(JsonException)
    2. On JsonException: errors.Add InvalidJson, return (null, null, null)
    3. Check dto.Kind == "binary"; if not: errors.Add InvalidKind, return (null, null, null)
    4. Check dto.Fields is not null and not empty
    5. Map each FieldDto to BinaryContractNode via private MapField method
    6. For bits containers: map each BitFieldDto to BitFieldInfo
    7. For arrays: map element FieldDto to ArrayElementInfo, handle Count as int or string from JsonElement
    8. For enums: copy Values dictionary and Primitive
    9. For strings: copy Encoding
    10. For all: map ValidationDto to ValidationRules, map x-error JsonElement to SchemaErrorInfo, copy DependsOn, DisplayName
    11. Return (fields dictionary keyed by name, dto.Endianness, metadata record)

    Create ContractMetadata record: `internal sealed record ContractMetadata(string? Id, string? Name, string? Version, Dictionary<string, string>? DisplayName);`
