**4. Pass 2: Dynamic-offset fields (D-10, D-11)**
After Pass 1, if `pass2StartIndex >= 0`:

a. Compute running offset accumulator. Start from the byte position where fixed fields ended. Walk backwards from `pass2StartIndex - 1` to find the last fixed field's end position:
```csharp
int runningOffset = 0;
if (pass2StartIndex > 0)
{
    var lastFixed = OrderedFields[pass2StartIndex - 1];
    int lastFixedSize = ComputeActualFieldSize(lastFixed, offsetTable, parseNameToOrdinal);
    runningOffset = lastFixed.AbsoluteOffset + lastFixedSize;
}
```
Where `ComputeActualFieldSize` is a new private helper:
- For non-array fields: returns `node.Size`
- For fixed arrays: returns `(int)node.Count * node.ArrayElement.Size`
- For semi-dynamic arrays: reads count from offsetTable and returns `resolvedCount * node.ArrayElement.Size`

b. Loop from `pass2StartIndex` to end of `OrderedFields`:
```csharp
for (int i = pass2StartIndex; i < OrderedFields.Length; i++)
{
    var node = OrderedFields[i];
    // Override the load-time offset with the computed runtime offset
    int actualOffset = runningOffset;

    byte fieldType = GetFieldType(node.Type);

    if (node.Type == "array")
    {
        // Same array expansion logic as Pass 1, but using actualOffset instead of node.AbsoluteOffset
        // ... resolve count (fixed or semi-dynamic) ...
        // ... clamp by available bytes (D-05) ...
        // ... expand elements ...

        int elementSize = node.ArrayElement.Size;
        runningOffset += resolvedCount * elementSize;

    else if (fieldType == FieldTypes.Padding)
    {
        runningOffset += node.Size;
```

**7. OffsetTable capacity for semi-dynamic arrays**
Semi-dynamic array element counts are unknown at the start of Parse(). The OffsetTable must accommodate them. Two options:
- Option A: Pre-allocate a generous capacity (e.g., TotalOrdinalCapacity + 256)
- Option B: Since OffsetTable.Set() silently ignores out-of-range ordinals, allocate with estimated capacity and accept that very large semi-dynamic arrays may have some elements only accessible via ArrayBuffer enumeration, not path lookup.
Use Option A with a reasonable buffer. The NameToOrdinal clone + ArrayBuffer still work correctly for all elements.
